# Commodity backstage management system install and start methods

## Environment depandencies:
```
install nodejs v6.12.3
install yarn
```

## Project initialization method:
install depandencies：  
```
yarn
```

## Dev mode run:
```
yarn run dev
```

## online packaging:
Mac / Linux sytem：
```
yarn run dist
```
Windows system：
```
yarn run dist_win
```
